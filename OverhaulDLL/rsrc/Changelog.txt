------------------------------------------------------------------------------------------
06/19/2018:
- New feature:
    - (Experimental) Ladder fix:
        Fixes the issue that caused players to fall through the map when playing with the
        framerate unlocked.

- New config settings:
    - LadderFix
	    Enables/disables ladder fix.
    - LadderFixRungIndexOverride
        Overrides the default rung index at which the player will exit a ladder while
        sliding down.

- New console commands:
    - ladder_fix
	    Enables/disables ladder fix.
    - ladder_fix_override
        Overrides the default rung index at which the player will exit a ladder while
        sliding down.

- New keybind:
    - ToggleLadderFix
	    Enables/disables ladder fix.



------------------------------------------------------------------------------------------
04/12/2018:
- Fixed an issue where the 4 Kings boss would sometimes become invincible if the player
  entered the boss arena with BossGuard anti-cheat enabled and an invader present.

01/13/2018:

- New features:
    - Aggressive AI challenge mod:
        Makes enemies extra aggressive, and aware of your presence from extremely long
        distances
    - Console can now be closed with the B button on the gamepad

- New config settings:
    - AggressiveAI
	    Enables/disables Aggressive AI challenge mod at startup
    - IoMonitorFilter
        Filters file I/O monitoring output to strings containing the specified filter
        string.

- New console commands:
    - challenge_aggressive_ai
	    Enables/disables Aggressive AI challenge mod
    - overhaul_build
        Prints build title and compile time for the current build of the Dark Souls
        Overhaul project

- New keybinds:
    - ToggleChallengeAggressiveAI
	    Enables/disables Aggressive AI challenge mod

- Fixes:
    - Fixed issue when not using custom save file path that could result in the current
      save file being overwritten.
    - Fixed file I/O monitoring with non-custom archive file paths
    - Fixed in-memory parameter editing for modified param files


------------------------------------------------------------------------------------------
01/09/2018:

- New features:
    - Black Phantom Enemies challenge mod:
        Turns all enemies (and bosses) into Black Phantoms, meaning they appear with a
        red glow and receive a stat buff (2x HP, 1.8x attack, and 1.2x defense).
    - Gravelord Phantoms challenge mod:
        Auto-spawns additional Gravelord Phantom enemies, even when the player has not
        been cursed by another player from the Gravelord covenant.
    - New configuration file section: "Game Input"

- New config settings:
    - BlackPhantomEnemies
	    Enables/disables Black Phantom Enemies challenge mod at startup
    - BlackPhantomEnemyDrawType
        Sets the draw type (phantom color) for NPCs affected by the "Black Phantom
        Enemies" challenge mod at startup
    - GravelordPhantoms
        Enables/disables Gravelord Phantoms challenge mod at startup
    - MouseInput
        Enables/disables mouse input at startup
    - ConsoleLockCamera
        Enables/disables locking camera angle when console is open

- New console commands:
    - challenge_black_phantom_enemies
	    Enables/disables Black Phantom Enemies challenge mod
    - challenge_black_phantom_enemy_draw_type
        Sets the draw type (phantom color) for NPCs affected by the "Black Phantom
        Enemies" challenge mod
    - challenge_gravelord_phantoms
	    Enables/disables Gravelord Phantoms challenge mod
    - gravelord_phantoms_despawn
        De-spawns existing Gravelord phantoms, but only if current character is in
        their first playthrough (NG+0)
    - mouse_input
        Enables/disables mouse input
    - camera_lock_when_console_open (and alias: console_lock_cam)
        Enables/disables locking camera angle when console is open

- New keybinds:
    - ToggleChallengeBlackPhantomEnemies
	    Enables/disables Black Phantom Enemies challenge mod
    - ToggleChallengeGravelordPhantoms
	    Enables/disables Gravelord Phantoms challenge mod
    - DespawnGravelordPhantoms
        De-spawns existing Gravelord phantoms, but only if current character is in
        their first playthrough (NG+0)
    - ToggleMouseInput
        Enables/disables mouse input
    - ToggleConsoleLockCamera
        Enables/disables locking camera angle when console is open

- Changes:
    - Renamed the following configuration file sections:
        - "Dark Souls Overhaul HUD" to "HUD"
        - "Dark Souls Overhaul Anti-Cheat" to "Anti-Cheat"

01/08/2018:

- New features:
    - Basic implementation of infinite save slots
    - XInput hooking (allows reading and modifying controller input)

- New config settings:
    - CheatsWarning
	    Enables/disables permanent "Cheats enabled" warning when player enables cheats

- New console commands:
    - save_file_create
	    Creates a new save file with 10 empty character slots (Existing save files are
        not modified)
    - save_file_index
        Sets the current save file index (must be called while viewing saved characters
        menu)
    - save_file_next
        Sets the current save file index to the next save file (must be called while
        viewing saved characters menu)
    - save_file_prev
        Sets the current save file index to the previous save file (must be called while
        viewing saved characters menu)

- New keybinds:
    - NextSaveFile
	    Sets the current save file index to the next save file (only works when viewing
        saved characters menu)
    - PrevSaveFile
        Sets the current save file index to the previous save file (only works when
        viewing saved characters menu)

- New controller button functions:
    - Left bumper and left D-pad button now switch to previous save file index when
        viewing saved characters menu
    - Right bumper and right D-pad button now switch to next save file index when
        viewing saved characters menu
		
- Updates:
    - When cheats are enabled, player will now be switched to a dedicated multiplayer
      network with other players who also have cheats enabled.

12/30/2017:

- New features:
    - BinocsTriggerBlock Anti-cheat:
        Prevents hackers from triggering the binoculars item-use effect on hit, leaving
        their victim defenseless.
    - DragonTriggerBlock Anti-cheat:
        Prevents hackers from triggering dragonification of a victim's head/body on hit.

- New config settings:
    - BinocsTriggerBlock
	    Enables/disables BinocsTriggerBlock anti-cheat on startup.
    - DragonTriggerBlock
	    Enables/disables DragonTriggerBlock anti-cheat on startup.
    - MonitorBdtFiles
	    Enables/disables file I/O monitoring for the BDT archive files.
    - MonitorBhdFiles
        Enables/disables file I/O monitoring for the BHD5 archive header files.
    - MonitorSaveFile
        Enables/disables file I/O monitoring for the save file.

- New console commands:
    - ac_binocs_trigger_block
	    Enables/disables BinocsTriggerBlock anti-cheat
    - ac_dragon_trigger_block
	    Enables/disables DragonTriggerBlock anti-cheat

- New keybinds:
    - ToggleBinocsTriggerBlockAC
	    Toggles BinocsTriggerBlock anti-cheat
    - ToggleDragonTriggerBlockAC
	    Toggles DragonTriggerBlock anti-cheat
		
- Fixes:
    - Fixed anomalous issues with a specific type of in-line assembly dereference

- Updates:
    - Update DsGameFile library & resource files
