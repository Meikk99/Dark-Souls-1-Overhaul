------------------------------------------------------------------------------------------
01/08/2018:

- New features:
    - Basic implementation of infinite save slots

- New config settings:
    - CheatsWarning
	    Enables/disables permanent "Cheats enabled" warning when player enables cheats

- New console commands:
    - save_file_create
	    Creates a new save file with 10 empty character slots (Existing save files are
        not modified)
    - save_file_index
        Sets the current save file index (must be called while viewing saved characters
        menu)
    - save_file_next
        Sets the current save file index to the next save file (must be called while
        viewing saved characters menu)
    - save_file_prev
        Sets the current save file index to the previous save file (must be called while
        viewing saved characters menu)

- New keybinds:
    - NextSaveFile
	    Sets the current save file index to the next save file (only works when viewing
        saved characters menu)
    - PrevSaveFile
        Sets the current save file index to the previous save file (only works when
        viewing saved characters menu)
		
- Updates:
    - When cheats are enabled, player will now be switched to a dedicated multiplayer
      network with other players using cheats.

12/30/2017:

- New features:
    - BinocsTriggerBlock Anti-cheat:
        Prevents hackers from triggering the binoculars item-use effect on hit, leaving
        their victim defenseless.
    - DragonTriggerBlock Anti-cheat:
        Prevents hackers from triggering dragonification of a victim's head/body on hit.

- New config settings:
    - BinocsTriggerBlock
	    Enables/disables BinocsTriggerBlock anti-cheat on startup.
    - DragonTriggerBlock
	    Enables/disables DragonTriggerBlock anti-cheat on startup.
    - MonitorBdtFiles
	    Enables/disables file I/O monitoring for the BDT archive files.
    - MonitorBhdFiles
        Enables/disables file I/O monitoring for the BHD5 archive header files.
    - MonitorSaveFile
        Enables/disables file I/O monitoring for the save file.

- New console commands:
    - ac_binocs_trigger_block
	    Enables/disables BinocsTriggerBlock anti-cheat
    - ac_dragon_trigger_block
	    Enables/disables DragonTriggerBlock anti-cheat

- New keybinds:
    - ToggleBinocsTriggerBlockAC
	    Toggles BinocsTriggerBlock anti-cheat
    - ToggleDragonTriggerBlockAC
	    Toggles DragonTriggerBlock anti-cheat
		
- Fixes:
    - Fixed anomalous issues with a specific type of in-line assembly dereference

- Updates:
    - Update DsGameFile library & resource files
